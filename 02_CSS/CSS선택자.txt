1  id 선택자       (#) :#id속성값
2 tag 선택자     (태그명): 태그명
3  class 선택자  (.)      : .class속성값
4  모든 요소 선택자  (* [asterisk] )  : *{}
  
5  기본 속성 선택자  : 선택자A [속성명='속성값']{}[속성명]   : [속성명='속성값']
6  자식 선택자( > , 1단계 하위)  : 부모 요소 > 자식 요소
7  후손 선택자(띄어쓰기, 하위 모두)  : 부모(조상)요소 후손요소 (공백넣음)
8  포커스가 맞춰졌을 떄 (:focus):  선택자A:focus (선택자A는 input요소)
9  체크가 되었을 떄(:checked)  : 선택자A:checked (선택자A는radio checkbox요소)
10  활성화 상태일 떄(:enaled) : 선택자A:enabled
11 비활성화 상태일 떄(:disabled) : 선택자A:disabled{} (선택자A는 input요소)
12  마우스를 올렸을 때(:hover)  : 선택자A:hover{}
13  클릭을 하고 있을 때(:active)  : 선택자A:active{}
14  바로 뒤 형제 선택(+)  : 선택자A + 선택자B
15  뒤에 있는 일치하는 형제 모두 선택(~, Tilde 틸드)  : 선택자A ~ 선택자B

    일반 구조 선택(위치 기준) -> 선택된 요소가 지정된 위치에 있는 요소가 맞으면 선택하는 선택자
16  선택된 요소가 첫 번째 요소인 경우 : 선택자A:first-child {}
17  선택된 요소가 마지막 요소인 경우  : 선택자A:last-child  {}
18  선택된 요소가 n번째 요소인 경우  : 선택자A:nth-child(n)  {} //n은 정수 혹은 수열

    형태 구조 선택자 -> 선택된 요소가 지정된 요소 중 n번째 요소가 맞으면 선택
19  같이 선태된 형제들 중 첫 번째 요소  : 선택자A:first-of-type
20  같이 선택된 형제들 중에서 마지막 요소 : 선택자A:last-of-type
21  같이 선택된 형제들 중에서 n번째 요소  : 선택자A:nth-of-type(n)      //앞에서부터
                                            선택자A:nth-last-of-type(n) //뒤에서부터
22  부정 선택자(:not()) -> 특정 요소만 빼고 선택  : 선택자A:not(선택자B)  //선택된 A 중에서 B만빼고
    
23  여러 선택자 동시 작성 : 선택자A, 선택자B
24  여러 클래스를 가진 요소 선택  : 클래스명A.클래스명B   //공백없이!
25  특정 태그 중 특정 클래스를 가진 요소 선택 : 태그명A.클래스명A
