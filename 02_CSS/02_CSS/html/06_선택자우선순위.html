CSS 선택자 우선 순위

    CSS는 기본적으로 style 태그 또는 .css 파일에 작성된 순서(위->아래) / 왼쪽->오른쪽) 방향으로 해석이 진행되지만,
    같은 요소에 여러 CSS 속성이 설정되는 경우 위치, 선택자 등에 따라서 우선순위가 결정된다.

    
      [꼭 알고 있어야 하는 것!]
    

    1) 동일한 우선 순위로 CSS 속성이 설정된 경우 ==> 제일 마지막에 작성된 CSS 속성이 반영된다(덮어쓰기?)

    2) 여러 선택자를 이용해서 같은 요소를 선택하더라도 지정된 CSS 속성이 다른 경우 모두 반영된다

    

    (1순위 우선 순위가 제일 높음(최우선))
    1순위 : !important

    2순위 : inline-style(html 요소에 직접 작성)   

    3순위 : id 선택자 (#id 속성 값)

    4순위 : class 선택자 (.class)

    5순위 : tag 선택자 (tag)
