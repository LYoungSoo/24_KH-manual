a(Anchor : 연결) 태그

    - 현재 HTML 문서와 다른 HTML 문서를 연결시켜 클릭 시 이동할 수 있게 만드는 태그

    - a 태그로 감싸진 글/내용 == 하이퍼 텍스트

    - 하이퍼 텍스트는 다른 문서와 구분되는 HTML 만의 핵심 기능(이었다가 word를 시작으로 다 따라함...)

a 태그 테스트

    - 속성 2가지
      1) href(Hypertext REFerence) : 연결할 주소를 작성하는 속성
      2) target = "_blank" : 새 탭으로 열기 ((구)새 창으로 열기)
