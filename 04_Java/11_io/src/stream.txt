Stream 클래스

입출력 장치에서 데이터를 읽고 쓰기 위해서 자바에서 제공하는 클래스
모든 스트림은 단방향(일방통행)이며, 각각의 장치마자 연결할 수 있는 스트림 존재
하나의 스트림으로 입출력을 동시에 수행할 수 없으므로 동시에 수행하려면 2개의 스트림(입력/출력) 필요

바이트 기반 스트림 (1 Byte) : InputStream (입력 스트림) / OutputStream(출력 스트림)
    ==> 한글과 같은 2바이트 문자는 들어가면 잘림.
    그래서 인터넷 주소창에 한글있는거 복사하면 특수문자로 다 바뀌어버리는게 그러한 원인때문인건가
문자 기반 스트림 (2 Byte)   : Reader (입력 스트림) / Writer (출력 스트림)
    ==> 2바이트 단위이므로 앵간한 문자 다 됨. 인터넷 채팅 메모장 등등에 다 씀
